## ■ Weekly Practice

### Week 02
- 변수 및 자료형 학습
- Scanner를 이용한 입력 처리
- 자동/수동 형변환 실습
- 산술, 논리, 증감 연산자 동작 확인
- 문자형(char) 및 Unicode 출력 실습

### Week 03
- 증감 연산자 및 다양한 연산자 학습
- 형변환과 오버플로우 확인
- if~else 조건문 실습
- 짝수/홀수 판별 및 문자 변환 프로그램 작성

### Week 04
- switch문과 조건문을 이용한 프로그램 작성
- for, while, do-while 반복문 학습
- 중첩 반복문을 활용한 구구단과 별 찍기 출력
- break, continue를 이용한 반복 흐름 제어

### Week 05
- 1차원/2차원 배열 학습
- 배열과 반복문을 이용한 데이터 처리
- 합계, 평균, 최댓값, 최솟값 계산 실습
- Random을 이용한 난수 및 중복 제거 구현

### Week 07
- 중간고사 대비 Java 기본 문법 복습
- Scanner 입력 및 반복문 활용 문제 풀이
- 클래스와 객체 생성 기초 학습
- Car, Card, Student, Tv 등의 클래스 구현 및 메서드 실습

### Week 09
- private 멤버변수와 캡슐화 학습
- 객체지향 방식으로 클래스 구조 개선
- 인스턴스 변수와 static 클래스 변수의 차이 학습
